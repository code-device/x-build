## 快速上手

x-build 是一个基于 [Node.js](http://nodejs.cn/) 环境的命令行工具。

```bash
npm install x-build -g
```

通过以下指令[创建](/Guide/Overview/Create.md)一个新项目，并且会自动帮你安装所有依赖：

```
x-build create [name]
```
