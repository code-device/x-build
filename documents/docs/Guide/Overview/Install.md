# 安装 CLI

## 准备

你需要在本地安装 node 和 git。

## 安装

配置好 NPM 源后，通过下面指令安装 x-build:

```
npm install x-build -g
```

mac 用户需要权限增加 sudo。

使用方式类似于 @vue/cli，提供了终端里的 mapwhale 命令。

## 相关工具

- [Vue Dev-Tools](https://github.com/vuejs/vue-devtools): Vue 在浏览器下的调试工具。

- [Vetur](https://github.com/vuejs/vetur): VS Code 插件。
